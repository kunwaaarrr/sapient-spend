Sapient Spend — YNAB user-flow reference

Every distinct screen from the recording, grouped by app section, plus a decision map of how they connect. Frames were taken only where the video is temporally still, so nothing is mid-animation. This set now also folds in four follow-up recordings — custom Views, Move Money, category Targets and Settings. Click any screen to open it full-resolution (native 1080×2340).

How the Assigned / Available columns show: they are not a separate screen — it’s a toggle. Open the ⋮ menu → Progress Bars. When Progress Bars is ON the budget shows one “Available to Spend” column with a bar under each row (screen 1). Turn Progress Bars OFF and it switches to the two-column Assigned + Available numeric view (screen 2). That is the button you pressed.
